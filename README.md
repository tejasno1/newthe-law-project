# The Law Project

A modern, responsive Next.js website with two stunning homepage designs for an online education platform.

## Features

- **Two Complete Homepages**: Two distinct designs based on the provided PDF mockups
- **Fully Responsive**: Mobile-first design that works on all devices
- **Smooth Animations**: Framer Motion animations for scroll-triggered reveals
- **Interactive Components**: FAQ accordions, mobile navigation, hover effects
- **Modern Stack**: Next.js 14, React 18, TypeScript, Tailwind CSS

## Pages

- `/` - Landing page with links to both homepages
- `/page1` - Dark theme with purple accents (Eduverse-style)
- `/page2` - Modern dark/light theme (Educore-style)

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
